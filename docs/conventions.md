# Accounting Conventions

Every convention below is implemented in `src/accounting.py` and applies
uniformly to all tables in Working Paper 2026.01. Where the literature
admits more than one treatment, the choice is stated and justified.

## 1. Earnings measure

Comprehensive income attributable to owners of the parent, as reported in
the statement of comprehensive income (DRA) of the consolidated financial
statements. Net income is retained as a robustness input.

Comprehensive income is required rather than net income because the
residual earnings model rests on the clean surplus relation. Items routed
through other comprehensive income — cash flow hedge reserves, foreign
currency translation adjustments — change book value without passing
through net income; excluding them would break the identity linking
earnings, dividends, and book value.

## 2. Return on common equity

    ROCE_t = Comprehensive Income_t / B_{t-1}

Beginning-of-period common shareholders' equity. Penman (2013, Box 5.1,
p. 147) states this definition first and illustrates it with the Nike
computation; the same box notes that ROCE for a full year is often
computed on average book value as a practical approximation.

The beginning-of-period base is used here for two reasons. First, it is
the base on which the residual earnings capital charge is levied, so it
is the only convention under which the identity

    RE_t = (ROCE_t − r) × B_{t-1}

holds exactly rather than approximately. Second, it is the convention of
the valuation chapters; the average-equity measure belongs to ratio
analysis (Penman, 2013, Chapter 12), where the object of measurement is
period profitability rather than the return demanded of capital in place.

ROCE is undefined when opening book value is zero or negative; the code
returns a missing value rather than a number in those cases.

## 3. Residual earnings

    RE_t = Comprehensive Income_t − r × B_{t-1}

equivalently, in spread × base form,

    RE_t = (ROCE_t − r) × B_{t-1}

The code computes the dollar form and verifies the spread × base
identity to within 1e-9 on every firm-year.

## 4. Clean surplus

    B_t = B_{t-1} + CI_t − d_t

where `d_t` denotes net distributions to shareholders: dividends plus
interest on equity plus share repurchases, less capital contributions and
issuances. A positive `d_t` is a net outflow to shareholders.

The identity is verified for every firm-year in the panel.
`reconcile_equity` flags any observation whose reported closing equity
departs from the implied value by more than the stated tolerance;
exceptions are documented before the observation is used.

On a per-share basis the identity reads

    BPS_t = BPS_{t-1} + EPS_t − DPS_t

and holds exactly when the share count is constant. Small residual gaps
arise when shares outstanding change during the period; these are
reported rather than adjusted away.

## 5. Required return

Ten percent throughout, applied uniformly to the three firms. The paper's
Section 3.3 sets out the reasoning: Penman states a cost of equity in
each canonical case and applies it directly, without firm-specific asset
pricing; the exercise is a demonstration of pricing logic rather than a
claim to valuation precision.

## 6. Growth in continuing value

Three scenarios (0, 1, 2 percent) are reported rather than a single
estimate. The growth rate in the continuing value is the average long-run
growth rate of residual earnings beyond the forecast horizon, bounded
above by the long-term nominal growth rate of the host economy.

## 7. Units

All rates are decimals in the code (0.10, never 10) and percentages in
the paper. The code rejects rates outside the interval [0, 1] to prevent
the percent/decimal error. Per-share figures are in Brazilian reais,
adjusted for stock splits and bonus issues as documented in the paper.

## 8. Audit rule

No number enters the canonical dataset without a reference to the filing
page from which it was transcribed, and no table in the paper contains a
number typed by hand. Every published figure is regenerated by code from
the canonical dataset.
